# IntegradorFinal2D_C1_Sosa_Infection
[BioContain Field Ops]
🎮 Descripción del Juego (Español)
BioContain Field Ops es un juego 2D top-down donde el jugador debe defender un objetivo central de oleadas de infectados.

El juego está diseñado con una arquitectura modular utilizando ScriptableObjects, un GameManager centralizado y enemigos configurables mediante datos externos.

## 🎮 Cómo jugar

- Moverse: WASD
- Apuntar: Mouse
- Disparar: Click izquierdo
- Objetivo: Defender el núcleo central de las oleadas de infectados.

Los enemigos (Swarmers) se desplazan en grupo hacia el jugador y el objetivo, generando presión constante.
Si el núcleo pierde toda su vida, el jugador pierde la partida.

🎮 Game Description (Spanish)
BioContain Field Ops is a 2D top-down game where the player must defend a central objective from waves of infected.

The game is designed with a modular architecture using ScriptableObjects, a centralized GameManager, and enemies configurable via external data.

## 🎮 How to Play

- Move: WASD
- Aim: Mouse
- Shoot: Left Click
- Objective: Defend the central core from waves of infected.

The enemies (Swarmers) move in groups toward the player and the objective, generating constant pressure.

If the core loses all its health, the player loses the game.
